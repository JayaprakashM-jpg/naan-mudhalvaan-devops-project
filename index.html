About This Project

🎯 Objective

To demonstrate a complete DevOps CI/CD pipeline using Jenkins, Git, and GitHub for automated web application deployment.

🛠️ Technologies Used

  • Jenkins - CI/CD Automation
  • Git & GitHub - Version Control
  • Maven - Build Tool
  • HTML/CSS/JS - Frontend
  • Tomcat - Application Server

⚙️ Pipeline Stages

  • Checkout - Pull code from GitHub
  • Build - Compile with Maven
  • Test - Run unit tests
  • Deploy - Deploy to server
