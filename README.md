# 📚 Naan Mudhalvan DevOps Project

## Student Management System with Jenkins CI/CD Pipeline

---

### 🎯 Project Overview

This project demonstrates a complete **DevOps CI/CD pipeline** for automating the build, test, and deployment of a web application. It was created as part of the Naan Mudhalvan program at Anna University.

| **Component** | **Technology** |
|--------------|----------------|
| Frontend | HTML5, CSS3, JavaScript |
| Build Tool | Apache Maven |
| Version Control | Git & GitHub |
| CI/CD | Jenkins Pipeline |
| Web Server | Python HTTP Server / Apache Tomcat |
